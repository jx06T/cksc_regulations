## 編輯身分與條件
> 任何人皆可提出修改要求

## 可修改狀況
1. 針對格式的錯誤修正
2. 針對內容可能的錯誤加上註解
3. 提供其他附件下載連結

## 不可修改狀況
1. 法條內容修改
2. 附件內容修改
> 若為以上兩情況，但認為有必要修改者請與班聯會反應

## 編輯方法
到要修改的頁面下方點擊編輯圖標，登入 github 後編輯文件並提出 pull request (PR)，會盡快檢查並決定是否接受

## 編輯技巧
1. 非必要請不要更改 Frontmatter 區塊
2. 格式更改請參考原有格式
3. 加入附註請使用以下指令
```
:::note
附註文字
:::
```
效果如下
:::note
附註文字
:::
除了 `note` 外還可以使用 `tip`、`caution`、`danger`
